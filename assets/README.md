# Tasko Website Assets

## Required Images

Please save the following images to this folder:

### 1. `screenshot.png`
- The main preview screenshot of the Tasko extension
- Used in the hero section browser mockup
- Recommended size: 1200×750 pixels or similar aspect ratio

### 2. `favicon-32.png` (Optional)
- 32×32 pixel PNG favicon
- Fallback for browsers that don't support SVG favicons
- Can be generated from logo.svg

### 3. `og-image.png`
- Open Graph image for social media sharing
- Size: 1200×630 pixels
- Shows when link is shared on Twitter, Facebook, etc.

## Already Included

- `logo.svg` - Main logo (checkmark icon with gradient)
- `favicon.svg` - SVG favicon
- `generate-assets.html` - Helper to generate PNG assets

## Screenshot Placeholders for Gallery

## How to Generate PNG from SVG

1. Open `generate-assets.html` in a browser
2. Right-click on the previews and save
3. Or use any online SVG to PNG converter
